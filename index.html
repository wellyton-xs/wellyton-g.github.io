comandos básicos:
git --help: obtém ajuda com vários exemplos de comandos
git status: obtém o status de modificações feitas.
git add: adiciona arquivos ao stage, preparando-os para o commit
git commit -m: cria um commit com os arquivos preparados (em stage)
git push: envia o commit para um repositório remoto
git pull: sincroniza alterações feitas no repositório remoto para que o nosso esteja atualizado.
git branch: mostra todas as branch's existentes.
git switch: cria ou seleciona uma branch.
git merge <nome_da_branch>: soma as alterações de uma branch
